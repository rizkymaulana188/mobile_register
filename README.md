# Complete Android Form Project

## Features

### 1. Complete Form
- TextInputLayout
- Name
- Email
- Password
- Confirm Password

### 2. Advanced Validation
- Empty validation
- Email format validation
- Password match validation
- Real-time validation

### 3. Selection Controls
- RadioGroup (Gender)
- Checkbox hobbies
- Minimum 3 hobbies validation

### 4. Spinner & Dialog
- Custom Spinner (City selection)
- AlertDialog confirmation on submit

### 5. Gesture Interaction
- Long Press button to reset form

### 6. Technology
- Kotlin
- Android Studio
- Material Design Components
